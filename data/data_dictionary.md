# Data Dictionary

## E-commerce Sales Analysis Dataset

### Dataset Overview
- **Total Records:** 1,000 products
- **Time Period:** 12 months
- **Categories:** 7 (Books, Clothing, Electronics, Health, Home & Kitchen, Sports, Toys)

### Column Descriptions

| Column Name | Data Type | Description | Example Values |
|------------|-----------|-------------|----------------|
| `product_id` | Integer | Unique identifier for each product | 1, 2, 3, ...  1000 |
| `product_name` | String | Name of the product | "Product_1", "Product_2" |
| `category` | String | Product category | "Books", "Electronics", "Clothing" |
| `price` | Float | Product price in USD | 10.44, 495.08, 234.56 |
| `review_score` | Float | Average customer rating (1.0 to 5.0) | 1.0, 3.5, 4.8, 5.0 |
| `review_count` | Integer | Total number of customer reviews | 100, 500, 999 |
| `sales_month_1` | Integer | Units sold in Month 1 (January) | 0 to 1000 |
| `sales_month_2` | Integer | Units sold in Month 2 (February) | 0 to 1000 |
| `sales_month_3` | Integer | Units sold in Month 3 (March) | 0 to 1000 |
| `sales_month_4` | Integer | Units sold in Month 4 (April) | 0 to 1000 |
| `sales_month_5` | Integer | Units sold in Month 5 (May) | 0 to 1000 |
| `sales_month_6` | Integer | Units sold in Month 6 (June) | 0 to 1000 |
| `sales_month_7` | Integer | Units sold in Month 7 (July) | 0 to 1000 |
| `sales_month_8` | Integer | Units sold in Month 8 (August) | 0 to 1000 |
| `sales_month_9` | Integer | Units sold in Month 9 (September) | 0 to 1000 |
| `sales_month_10` | Integer | Units sold in Month 10 (October) | 0 to 1000 |
| `sales_month_11` | Integer | Units sold in Month 11 (November) | 0 to 1000 |
| `sales_month_12` | Integer | Units sold in Month 12 (December) | 0 to 1000 |

### Calculated Fields (Created During Analysis)

| Field Name | Calculation | Description |

### Data Quality Notes

- **Missing Values:** None detected
- **Duplicates:** No duplicate product_ids
- **Outliers:** Some products have 0 sales in certain months (normal business variation)
- **Price Range:** $7.29 to $499.86
- **Review Score Range:** 1.0 to 5.0
- **Review Count Range:** 1 to 999

### Business Context

- **Months represent:** Calendar months (Month 1 = January, Month 12 = December)
- **Sales figures:** Unit sales (not revenue)
- **Review scores:** Customer ratings on 5-star scale
- **Categories:** Represent major product lines in the e-commerce store
